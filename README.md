# strapi-provider-upload-s3-custom-endpoint

Provides similar functionality to [strapi-provider-upload-s3](https://github.com/strapi/strapi/tree/master/packages/strapi-provider-upload-aws-s3) with a custom endpoint.  Instead of selecting a region, you can enter your endpoint URL to use any s3 compatible / interoperable service, such as:

* [Google Cloud Storage](https://cloud.google.com/storage/docs/interoperability)
* [Wasabi](https://wasabi.com/)
* [Linode](https://www.linode.com/2019/10/30/linode-object-storage-is-here/)
* [Digital Ocean](https://www.digitalocean.com/products/spaces/)
